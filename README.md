# lrEbs
Yet an other Biblioteca System, designated for archeologs and other humanitarioan studies
## Getting Started
* Prerequisites (required software, dependencies):
	+ Install PHP 8.4.16 and the Laravel framework v12.
	+ Set up your local development environment with Inertia.js v2 for React client-side applications.
   

* Installation instructions:
  1. Clone this repository using Git or download it as a ZIP file.
  2. Run `composer install` to install dependencies, including inertiajs and react starter kit.
  3. Configure your database settings in the `.env` file to enable Laravel Eloquent ORM functionality.
  4. Run `php artisan migrate` to create the necessary database tables for Laravel Eloquent ORM.
  5. Install required npm packages by running `npm install`.
  6. Start the development server with `npm run dev`.
  7. Access the application at <http://localhost:8000>.
